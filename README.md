<h1>Memory Game</h1>


## Como jogar

<p>Para jogar o jogo, basta clicar em uma das cartas e depois em outra. Se as cartas forem iguais, elas permanecerão abertas. Se não forem iguais, elas se fecharão automaticamente.</p>
